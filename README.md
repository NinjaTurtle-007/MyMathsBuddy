## 🎯 **What is MathBuddy?**

MathBuddy is a **FREE** web-based tool that generates unlimited CBSE-aligned mathematics practice worksheets for classes 1-8. No sign-up, no data collection, no hassle!

**Born from frustration:** As a working mom tired of midnight worksheet hunts, printer battles, and expensive photocopies, I built this tool to make exam preparation easier for families like mine.

---

## ✨ **Features**

✅ **Complete CBSE Coverage** - Classes 1-8, all major topics  
✅ **3 Difficulty Levels** - Easy, Medium, Hard  
✅ **Instant Generation** - Create worksheets in 30 seconds  
✅ **Unlimited Practice** - Every worksheet is unique with different numbers  
✅ **PDF Download** - Print or practice digitally  
✅ **Show/Hide Answers** - Check your work easily  
✅ **100% Free** - No subscriptions, no hidden costs  
✅ **Privacy-First** - No data collection, runs locally in browser  
✅ **Mobile Friendly** - Works on phones, tablets, laptops

## 🚀 **Quick Start**

### **Option 1: Use Online (Easiest)**

Simply visit: **[mymathsbuddy.pages.dev](https://mymathsbuddy.pages.dev)**

1. Select your class (1-8)
2. Choose a topic
3. Pick difficulty level
4. Click "Generate Worksheet"
5. Practice or download as PDF!

### **Option 2: Run Locally**

```bash
# Clone the repository
git clone [https://github.com/ninjaturtle-007/MyMathsBuddy.git](https://github.com/NinjaTurtle-007/MyMathsBuddy.git)

# Open the file
cd MyMathsBuddy
# Simply open index.html in your browser
# No installation needed!
```

## 🎓 **For Parents**

### **How to Use This Tool Effectively:**

1. **Supplement, Don't Replace**
   - Use alongside school homework
   - Not a substitute for classes
   - Always verify with school curriculum

2. **Supervise Usage**
   - Sit with younger children
   - Check answers together
   - Use as learning opportunity

3. **Verify Answers**
   - Always double-check generated answers
   - Report any errors found
   - Teach kids to think independently

4. **Encourage Regular Practice**
   - 15-20 minutes daily is better than 2 hours before exam
   - Mix difficulty levels
   - Cover different topics

5. **Don't Stress!**
   - This is practice, not a test
   - Mistakes are learning opportunities
   - Focus on understanding, not just answers
